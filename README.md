# AI-Play Front-Web

Project AI Play의 Web App UI를 구현하는 Repo

## Stack

- React
- TailwindCSS

## 준비 사항

```
// package.json에 있는 것들 설치
npm i
```

> danfojs 자세한 적용 과정은 아래 링크 참고
> [React - Danfo.js 적용 과정](https://github.com/AI-Play/Front-Web/discussions/22#discussioncomment-2215525)

## 개발 서버 실행

```
npm start
```
